## View

- 스프링 부트 thymeleaf viewName 매핑의 기본은 `resources:templates/ +{ViewName}+ .html`
- `spring-boot-devtools`라이브러리를 추가하고 html 파일을 컴파일 해주면 서버 재시작 없이 View 파일 변경이 가능(Intellij Menu > build > Recompile)
